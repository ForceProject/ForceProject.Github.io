Template

The following template for creating your own scripts has includes all of the built in server delegate calls. Create a new file and copy and paste the code below to get started.
You are free to rename the class and the instance name of the Force Server below.
To change the port being used by the server, change 5555 on line 28 to whatever port you want, just be sure that the system is not already using that port.
It is also possible to change the number of clients that can connect to the server at any given time, by default it is 5, to change it, change the 5 on line 28 to whatever number you want greater than 1.
