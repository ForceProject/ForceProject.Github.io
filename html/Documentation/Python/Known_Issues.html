Built-In Functions

The Python Force Server is designed to be run on almost any computer with networking capabilities.

Sending Data

Data can be sent from your script to your client(s) very easily. Do this by calling a function built into ForceServer like below:

Function Declaration

def sendMessage(self, deviceName, tag, value):

Function Usage

# To a certain device
self.<Instance Name of Force Server>.sendMessage("Device Name", "Target Tile Tag", <Data>)
# To all devices
self.<Instance Name of Force Server>.sendMessage(None, "Target Tile Tag", <Data>)

Note that <Data> can be any data type, the data type should correspond to the data type of the tile you are sending data to.
